# E-COMMERCE-BACKEND

## Project Overview

This project implements the backend of an e-commerce platform using Node.js, Express.js, and SQLite. The system provides APIs for user authentication, product management, and order processing.

## Objective

To develop a backend system capable of managing users, products, and customer orders through RESTful APIs connected to a database.

## Features

* User Registration and Authentication
* Product Management
* Product Listing
* Order Processing
* Database Integration
* REST API Architecture

## Technologies Used

* Node.js
* Express.js
* SQLite
* JavaScript

## API Endpoints

### User Registration

POST /register

### Add Product

POST /products

### Get Products

GET /products

### Create Order

POST /orders

## Database

SQLite database is used to store user, product, and order information.

## Deliverable

A functional backend API supporting authentication, product management, and order processing with database connectivity.

## Conclusion

This project demonstrates backend development concepts for an e-commerce platform using Express.js and SQLite. The system provides scalable APIs for handling users, products, and orders.
